# IceSculpterVR
A voxel-based ice sculpting game in VR! Made for Oculus Quest 

## Goals 
- Implement a voxel grid with an ice texture & shader and some kind of mesh interpolation.
- Add a 2-handed fire sword XR interactable for slicing ice.
- Add an ice blaster for adding more ice!
- Add an option to export Ice sculpture as a 3D mesh (I think this counts as logging?)
